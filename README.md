# BLOG
## Blog简介
这个网站是由几位软件工程学院的学生创建的个人博客，专门为软件工程学院的同学们、老师们提供一个交流、学习和资源分享的平台。  
项目的主要目标是为软件工程学院的学生提供一个定制化的平台，以满足他们在学术和学习方面的需求。通过提供与软件工程相关的文章、题库、学习资源和选课指南，希望帮助学生们更好地理解和应用软件工程知识，并促进他们的学术发展。  
 
## Blog功能介绍
* Blog文章
* 软工题库——答题测试
* 匿名提问箱
* 软工学习资源库
* 软工选课通
* 后台管理
